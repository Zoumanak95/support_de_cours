AIDE MEMOIRE DES SELECTEURS CSS DEJA VU :

  • * : l'astérisque sélctionne tous les élément de la page HTML
  • .Ma-jolie-nav : Selectionne la ou les balses ayant la class "ma-jolie-nav"
  • #Mon-unique-id : Selectionne la ou les balses ayant l'id "mon-unique-id"
  • DIV, ARTICLE, ASIDE : Selectionne toutes les balises <div>, <article> et <aside>
  • NAV A : Selectionne tout les liens inclus dans la balise <nav>
  • NAV A:HOVER : Selectionne tous les liens de la NAV et active une animation au survole de la souris

Proporétés CSS utilisées dans cette page

Décrivez leur usages et les valeurs possibles en fonction de ce aue vous avez deja vu

  1. BACKGROUND : Propriétés de l'arrière-plan.
  2. COLOR : .La propriété color définit la couleur de premier plan d'un élément texte et de ses éventuelles décorations.
  3. FLOAT : La propriété float indique qu'un élément doit être retiré du flux normal et doit être placé sur le côté droit ou sur le côté gauche de son conteneur
  4. FONT-FAMILY : Permet de choisir une police d'un text
  5. FONT-SIZE : Permet de choisir une taille de police plus précise
  6. HEIGHT : Hauteur de l'élément.
  7. LEFT : Permet de déplacer un élément vers la gauche
  8. LINE-HEIGHT : Hauteur de ligne
  9. MARGIN : Marges extérieures de l'élément.
  10. PADDING : Marges intérieures de l'élément
  11. POSITION : Positionnement de l'élément.
  12. TEXT-ALIGN : Détermine l'alignement horizontal du texte à l'intérieur d'un élément.
  13. TEXT-DECORATION : Soulignement ou rayure du texte.
  14. TOP : Permet de fair monter l'élément
  15. TRANSITION : precise un parametre transition est un raccourci qui permet de spécifier l'ensemble des propriétés CSS de transition sur lesquelles on veut effectuer une transition CSS
  16. WIDTH : Permet de donné une largeur a des éléments

Propriétés CSS à rechercher

Rechercher les propriétés ci-dessous, faite des test avec cette page depuis l'inspecteur de page (console de debuggage) F12 sur firefox.

  1. FONT-WEIGHT : Mise en gras ou extra-gras (extra-bold).
  2. LETTER-SPACING : Augmente ou diminue l'espace entre les caractères dans un texte.
  3. WORDS-PACING : Propriété augmente ou diminue l'espace blanc entre les mots.
  4. TEXT-SAHDOW : Ajoute des ombres au texte.
  5. BACKGROUND-IMAGE : Permet de mettre une image en fond
  6. BACKGROUND-SIZE : Permet de modifier la taille du background
  7. OPACITY : Permet de définir l'intensité d'un élément comme le rendre plus transparent
  8. BORDER-STYLE : Donnne un style a la bordure d'un élément
  9. BOX-SHADOW : Permet de rajouté de l'ombrage a une box
  10. LIST-STYLE-IMAGE : Permet de mettre une image à la place du marqueur d'item de liste défini par list-style-type
  11. BORDER : Rajoute une bordure
  12. BORDER-RADIUS : Permet de définir des coins arrondis pour la bordure d'un élément.
  13. BORDER-WIDTH : Permet de donné une largeur a une bordure
  14. FONT-VARIANT : Permet de spécifier que le texte soit en petites capitales dans un élément
  15. BACKGROUND-COLOR : Couleur de l'arrière-plan.
  16. !IMPORTANT : Permet de spécifier que la propriété est la plus importante dans votre feuille en lui donnant un poids plus important.
